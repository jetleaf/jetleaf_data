Returns a new Slice<U> by applying the mapper function to each element.

This is useful for mapping the content to a different type while retaining slice metadata (page number, size, sorting, navigation).

Example

final idSlice = slice.map((user) => user.id);
print(idSlice.getContent()); // List<int> of user IDs

Implementation

@override
